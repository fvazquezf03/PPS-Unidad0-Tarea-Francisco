# PPS-Unidad0-Tarea-Fran
# Nombre del Proyecto
PPS Unidad 0 - Francisco Vázquez

## Descripción
Este proyecto documenta la creación y despliegue de documentación usando Git, GitHub Actions, MkDocs y Docker.  
Incluye:
- Creación de un repositorio Git.
- Configuración de workflow de GitHub Actions para generar la documentación.
- Publicación de la documentación mediante GitHub Pages y NGINX con Docker.

## Contenido del repositorio
- `docs/` - Carpeta principal de la documentación MkDocs (opcional según configuración).
- `index.md` - Introducción y menú de la documentación.
- `git.md` - Documentación de la creación del repositorio Git.
- `gitActions.md` - Documentación del workflow en GitHub Actions.
- `docker.md` - Documentación del despliegue de NGINX con Docker.
- `conclusiones.md` - Conclusiones de la actividad.
